# AgentGraph App Naming

AgentGraph is the current product and window title for this repository.

- `index.html` sets the browser document title to `AgentGraph`.
- `src-tauri/tauri.conf.json` sets both `productName` and the main window `title` to `AgentGraph`.
- The old `D3 Force Graph` label may still appear in historical node titles, but runtime-facing app naming should use AgentGraph.

This naming context updates [[AgentGraph Project Overview]] and belongs with the UI polish summarized in [[Conversation Summary - AgentGraph Search And Filter Polish]].
